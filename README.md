# manage to coupon webpage
>3학년 겨울방학에 진행하는 개인 프로젝트<br>

## project purpose
동아리 회원 정보를 입력받으면 DB에 저장하는 프로젝트
- 회원 정보 CRUD (진행 중)
- REST API json 파일로 문서화
- password 암호화 (Spring Security)

## develop environment
### JDK (Java Development Kit)
- Java 1.8.x

### Server (Controller)
- Spring Boot

### Database (Model)
- MySQL
- Lombok plugin
- JPA

### Front-End (View)
- html
